To build this project using Apache Ant:

1. Ensure you have Apache Ant installed and available in your PATH (or ANT_HOME enviroment)
2. Navigate to the project directory: E:\ApacheAntProjects\1
3. Run the following command to build all JAR files:
   ant build
4. To run the application after building:
   ant run

The build process will create the following JAR files in the dist/ directory:
- PepperMintLoader.jar
- PepperMintUI.jar
- PepperMintGens.jar
- PepperMint.jar
- PepperMintHater.jar
- PepperMintEncryption.jar
- PepperMintARL.jar
