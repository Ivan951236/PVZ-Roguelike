To verify an encrypted theme file:
Download and Install GpgFrontend
Run GpgFrontend
Locate the theme file
In this Repository, you can find a "PepperMintThemes" folder, this is where your themes will be
Go in this directory, then in this "IvyOS" directory
There is a .sig file
You can import the openpgp-key file
and verify the signature with this key

In the real situation, you would package it yourself in a tarball compressed in XZ with the extension .pmt
